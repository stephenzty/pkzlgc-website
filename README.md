# 常州市普克制冷新版官网

这是一个可直接部署到 Cloudflare Pages 的静态官网版本，信息与素材来自现站 `https://www.pkzlgc.com/`，并重新整理为更现代的企业展示页。

## 文件结构

- `index.html`：页面结构、SEO 元信息、联系方式、结构化数据
- `styles.css`：响应式视觉样式
- `script.js`：移动端导航、服务切换、滚动动效、邮件咨询表单
- `assets/`：从现站下载的 logo、首屏图、案例图、新闻图等素材

## 部署到 Cloudflare Pages

1. 把本目录推送到 GitHub 仓库。
2. 在 Cloudflare Pages 新建项目，连接该仓库。
3. 构建命令留空，输出目录填写 `.`。
4. 绑定新购买的域名。
5. 表单目前使用 `mailto:13806114401@139.com`，如果需要后台收集线索，可后续接入 Cloudflare Pages Functions 或第三方表单服务。

## 后续可优化

- 替换更高清的企业门店、施工现场、团队与荣誉资质照片。
- 如果确认新域名和备案信息，需要同步更新页脚备案与结构化数据。
- 如需多页面 SEO，可拆分为产品中心、案例中心、关于我们、新闻中心、联系我们等独立页面。
